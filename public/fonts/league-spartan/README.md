# League Spartan Font Files

Coloca aquí los archivos de fuente League Spartan en formato TTF o WOFF2.

## Archivos necesarios:

1. **LeagueSpartan-Regular.ttf** (o .woff2) - Peso normal (400), estilo normal
2. **LeagueSpartan-Italic.ttf** (o .woff2) - Peso normal (400), estilo itálico
3. **LeagueSpartan-Bold.ttf** (o .woff2) - Peso bold (700), estilo normal
4. **LeagueSpartan-BoldItalic.ttf** (o .woff2) - Peso bold (700), estilo itálico

## Cómo obtener los archivos:

1. Visita [Google Fonts - League Spartan](https://fonts.google.com/specimen/League+Spartan)
2. Descarga la fuente completa
3. Extrae los archivos TTF o WOFF2
4. Coloca los 4 archivos en esta carpeta con los nombres exactos indicados arriba

## Nota:

Si los archivos tienen nombres diferentes, actualiza las rutas en `components/reports/TiposCasosPDF.tsx`
